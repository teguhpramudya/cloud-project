# Cloud Computing Final Project

## Deploy Website Company Profile Using Docker and GitHub

### Technologies Used
- HTML
- CSS
- JavaScript
- Docker
- Nginx
- GitHub

---

## Run Project

### Build Docker Image
```bash
docker build -t cloud-project .
```

### Run Docker Container
```bash
docker run -d -p 8080:80 cloud-project
```
